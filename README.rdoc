This is a test app for Coub API (http://coub.com/dev/docs) described in this tutorial (in Russian): https://medium.com/@igorgl/туториал-по-coub-api-6bf7988d2c9d

You need to register your application on http://coub.com/dev/applications to use this application.

This app requires ffmpeg with freetype support, on Mac you can install it via Homebrew:

  brew install ffmpeg --with-fdk-aac --with-ffplay --with-freetype --with-frei0r --with-libass --with-libvo-aacenc --with-libvorbis --with-libvpx --with-opencore-amr --with-openjpeg --with-opus --with-rtmpdump --with-schroedinger --with-speex --with-theora --with-tools

Feel free to fork and clone and do whatever you want with this code.
